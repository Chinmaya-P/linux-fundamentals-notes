# File System Navigation


## What I learned

- Navigating the Linux filesystem using the terminal
- Identifying the contents of a directory, including files and sub-directories
- Finding hidden directories
- Printing out the contents of .txt files within a directory
- Using the Terminal to output text contents into a .txt file within a directory of my choosing.


## Commands and Examples

'''bash
 ls kali
 ls \home\kali\Desktop
 pwd 
 ls -la
 cd \home\kali\Documents
 echo "Hello World" > HelloWorld.txt
 cat HelloWorld.txt
 '''

## What Confused Me

- The idea of hidden directories
- Having to use the full filepath when not within the correct directory e.g. navigating from home/kali/Documents to home/kali/Desktop compared to just typing cd Desktop when at home/kali

## Reflections 

- I found over time, the use of the CLI to be more and more intuitive and I enjoy being able to navigate comfortably without having to rely on GUI.
