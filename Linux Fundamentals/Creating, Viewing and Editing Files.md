## What I learned  
-  How to create files from the CLI
- How to append text to files using the CLI
- How to use both CLI and external text editors to edit existing files
- How to output the contents of a file to the command line itself
  
## Commands and Examples  
```bash
touch newfile.txt
echo "Hello World" > newfile.txt
echo "How are you?" >> newfile.txt
nano newfile.txt
mousepad newfile.txt
cat newfile.txt
```
