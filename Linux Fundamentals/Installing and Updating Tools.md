## What I learned  
-  How to perform an OS update on Kali Linux using the CLI
- How to install new software on Kali Linux using the CLI
- Using git to clone into repositories and install tools from them
  
## Commands and Examples  
```bash
sudo apt update && sudo apt upgrade
sudo apt install cron-daemon-common
sudo git clone https://github.com/Dewalt-arch/pimpmykali.git


```
  
## What Confused Me  
-  The difference between update and upgrade initially (Then I realised update checks existing package lists for updates while upgrade installs the updated files)
  
## Reflections  
-  GitHub is a powerful source to acquire many tools and scripts that can be executed on a Linux machine, or any operating system. 
- I find it easier to identify what packages you install and update from CLI compared to GUI usage.
